# Development Roadmap

## Phase 1
- **Goals**: Setup, Docs, Core Skeletons, Auth (APIs).
- **Outcome**: A solid base repo running on docker-compose and handling DB creation.

## Phase 2
- **Goals**: Campaign management endpoints, Lead Models, Mock Providers.
- **Outcome**: Users can define parameters for campaigns and receive mocked leads logically.

## Phase 3
- **Goals**: LangGraph Architecture. Agent nodes mapped to qualifications and email drafting logic.
- **Outcome**: Agents generate email drafts and stop at the 'Approval Gate'.

## Phase 4
- **Goals**: UI implementations. Dashboard, approvals, detailed flows.
- **Outcome**: The web wrapper functions smoothly handling backend data.

## Phase 5
- **Goals**: Quality Assurance, Observability tools, Docker/CI pipeline finalization.
- **Outcome**: System runs flawlessly inside a containerized setup with unit/e2e tests passing and deploying.
