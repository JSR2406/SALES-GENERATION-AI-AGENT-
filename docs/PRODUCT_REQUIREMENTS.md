# Autonomous B2B Sales Agent

## Product Vision
A multi-tenant SaaS platform that helps small B2B startups and agencies automate outbound prospecting and personalized cold outreach.

## Core Features
- **Authentication**: Multi-tenant user isolation setup, user profiles.
- **Campaign Management**: CRUD operations, start/pause functionality, flexible sending mode (manual or auto-send).
- **Lead Discovery**: Dynamic integration with web/scraping providers to fetch role/industry specific leads.
- **Lead Qualification**: AI-based matching of leads against campaign ideal customer profiles (ICP).
- **Personalization Engine**: LLM-driven generation of subject, opening line, and email body using specific, non-hallucinated context.
- **Approval Workflow**: Manual QA pipeline vs auto-send processing.
- **Email Sending & Tracking**: Extendable email provider structure, reply classification (interested, bounced, etc).
- **Dashboard & Analytics**: Pipeline tracking, top-level stats, and campaign metrics.
- **Audit & Logs**: Traceable execution paths for LangGraph agents.
